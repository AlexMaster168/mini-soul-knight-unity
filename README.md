# Soul Knight Clone

Roguelike dungeon crawler в стиле Soul Knight, построенный на Unity. Процедурная генерация подземелий, 35+ видов оружия, 40+ типов врагов, система магазина, ловушки и уникальные анимации стрельбы.

## Запуск

### Быстрый запуск
```
Play.bat
```
Автоматически открывает Unity, загружает сцену и входит в Play Mode.

### Вручную
1. Открой Unity Hub
2. Открой проект `C:\Projects\soul-knight`
3. Открой сцену `Assets/Scenes/MainScene.unity`
4. Нажми Play

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| Мышь | Прицел |
| ЛКМ | Стрельба вперёд |
| ПКМ | Стрельба назад |
| Space | Рывок (Dash) |
| 1-5 | Быстрый выбор оружия |
| Q | Следующее оружие |
| E | Магазин (рядом с торговцем) |
| Tab | Статы персонажа |
| Escape | Закрыть меню |

## Характеристики игрока

- **HP**: 150
- **Энергия**: 200 (тратится на стрельбу, регенерация 5/сек)
- **Броня**: 0-100 (поглощает урон)
- **Скорость**: 8f
- **Рывок**: 15f, 0.2с, кулдаун 1с
- **Скорострельность**: 0.12с (быстро!)
- **Энергозатраты**: 3 за выстрел

## Золотая система

- Враги дропают **5-15 монет** при смерти (боссы: 50+ монет)
- Монетки притягиваются к игроку (магнит 5 юнитов)
- Золото тратится в магазине

## Магазин

Рядом с каждым торговцем нажми **E** чтобы открыть магазин. Управление: **стрелки/WS** — выбор, **Enter/H** — купить, **E/Esc** — закрыть.

| Предмет | Цена | Эффект |
|---------|------|--------|
| Random Weapon | 25G | Случайное оружие из каталога |
| Health +50 | 15G | Восстанавливает 50 HP |
| Energy +80 | 10G | Восстанавливает 80 энергии |
| Armor +30 | 20G | Добавляет 30 брони |
| Damage x2 (8s) | 18G | Удвоенный урон на 8 секунд |
| Speed x1.5 (8s) | 12G | Полтора скорости на 8 секунд |

- **Werewolf**: двойной удар
- **Berserker**: двойной урон
- **Lich**: призывает скелетов + кольцо из 4 пуль
- **Warlock**: 3 снаряда веером
- **StormMage**: кольцо из 7 молний
- **Wraith/Shadow**: один мощный снаряд
- **Nightmare**: вращающееся кольцо из 4 пуль
- **Harpy**: два снаряда веером
- **Troll**: регенерирует HP при ударе
- **Abomination**: шанс двойного удара
- **FireElemental**: огненный след + взрыв при смерти
- **LivingBomb**: огромный взрыв при смерти (50 урона)
- **Assassin**: тройной урон + рывок
- **CrystalGolem**: 300 HP, медленный но неуязвимый

## Боссы

- **CrownedBoar** (800 HP): веер из 5 пуль → 8 пуль + миньоны → кольцо из 12
- **Dragon** (1000 HP): дыхание + мины
- **Necromancer** (600 HP): призывает миньонов + магия

3 фазы боя с усилением при потере HP. После победы — **экран YOU WIN!** с торжественной музыкой.

## Подземелье

- 12-18 комнат на этаже
- Процедурная генерация сетки 5x5
- Комната зачищается → лут дропается + спавнится торговец
- Комната перед боссом: HolyGrenade + 50 брони
- Враги спавнятся при входе в новую комнату
- Сложность растёт с этажами (6 этажей)
- Ловушки-шипы (0-1 на комнату, 5 урона)

## Лут

### Предметы на полу
- **Аптечки** (сердечки): +50 HP
- **Энергия** (молнии): +80 энергии
- **Оружие** (мечи): случайное оружие
- **Усиление урона** (пламя): x2 на 8с
- **Усиление скорости** (крыло): x1.5 на 8с
- **Броня** (щиты): +30 брони
- **Монеты** (золотые круги): 1-3 золота

### Дроп-рейты
- Обычные враги: 25% HP, 45% энергия, 50% оружие
- Tank/DarkKnight: 40% HP, 55% энергия, 65% оружие
- Боссы: увеличенные дропы + 50 монет

## Интерфейс

- Полоса HP (красная) с текстом
- Полоса энергии (синяя) с текстом
- Полоса брони (голубая) с текстом

## Графика

### Процедурные спрайты
- Все спрайты генерируются кодом (16-64 px)
- Сердечки, молнии, мечи, щиты, монеты для лута
- Уникальные спрайты для каждого типа врага
- Спрайт торговца (фиолетовая мантия)
- Ловушки-шипы (металлические шипы на платформе)

### Визуальные эффекты
- Динамическое освещение (свет вокруг игрока, факелы)
- Тёмная виньетка по краям экрана
- Анимированное пламя факелов
- Частицы пыли в комнатах
- Светлячки (блуждающие огоньки)
- Тени под игроком и врагами
- Вспышки света при попаданиях
- Красная вспышка экрана при уроне
- Тряска камеры при стрельбе и уроне
- Детали на полу (камни, трещины)

### Уникальные анимации стрельбы
Каждое оружие имеет визуальные эффекты:
- Вспышки дула разных размеров и цветов
- Дым и огонь
- Гильзы (SMG)
- Трассеры и лучи (снайперки)
- Электрические искры (энергетическое)
- Магические искры (экзотическое)
- Дуги замаха (мили)
- Отдача и тряска камеры
