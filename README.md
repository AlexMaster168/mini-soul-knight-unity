# Soul Knight Clone

Roguelike dungeon crawler в стиле Soul Knight, построенный на Unity.

## Запуск

### Быстрый запуск
```
Play.bat
```
Автоматически открывает Unity, загружает сцену и входит в Play Mode.

### Вручную
1. Открой Unity Hub
2. Открой проект `C:\Projects\soul-knight`
3. Открой сцену `Assets/Scenes/MainScene.unity`
4. Нажми Play

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| Мышь | Прицел |
| ЛКМ | Стрельба |
| Space | Рывок (Dash) |
| Q | Смена оружия |
| Tab | Статы персонажа |
| ESC | Пауза |

## Характеристики игрока

- **HP**: 150
- **Энергия**: 200 (тратится на стрельбу, регенерация 5/сек)
- **Броня**: 0-100 (поглощает урон)
- **Скорость**: 8f
- **Рывок**: 15f, 0.2с, кулдаун 1с

### Этаж 1
Slime, Goblin, Bat, Spider

### Этаж 2
+ Skeleton, Wolf, Speedster

### Этаж 3
+ Shooter, Archer, Ghost, Zombie, Snake

### Этаж 4
+ DarkKnight, Mage, IceMage, FireMage, Orc, Flyer

### Этаж 5+
Tank, Guardian, Spawner, Golem, NecroMage

### Уникальные способности
- **FireMage**: 3 огненных шара
- **IceMage**: 3 ледяных снаряда
- **NecroMage**: кольцо из 5 пуль
- **Bomber**: взрывается при смерти
- **Spawner**: призывает миньонов
- **Ghost**: кружит вокруг игрока
- **Speedster**: быстрый раш
- **Tank/Golem**: нокбек при ударе

## Боссы

- **CrownedBoar** (800 HP): веер из 5 пуль → 8 пуль + миньоны → кольцо из 12
- **Dragon** (1000 HP): дыхание + мины
- **Necromancer** (600 HP): призывает миньонов + магия

3 фазы боя с усилением при потере HP.

## Подземелье

- 12-18 комнат на этаже
- Процедурная генерация сетки 5x5
- Комната зачищается → лут дропается
- Комната перед боссом: HolyGrenade + 50 брони
- Враги спавнятся при входе в новую комнату
- Сложность растёт с этажами

## Лут

- **Аптечки** (зелёные): +50 HP
- **Энергия** (синие): +80 энергии
- **Оружие** (жёлтые): случайное оружие
- **Усиление урона** (красные): x2 на 5с
- **Усиление скорости** (голубые): x1.5 на 5с
- **Броня** (голубые квадраты): +50 брони
- Враги дропают лут при смерти (25% HP, 20% энергия, 5% оружие)

## Интерфейс

- Полоса HP (красная)
- Полоса энергии (синяя)
- Полоса брони (голубая)
- Индикатор оружия
- Счёт, этаж
- Миникарта (правый нижний угол)
